How does Image Tinder work?

  • Select the source and destination folders with the buttons.
  • Use the keyboard arrows to move (→) or delete (←) images.
  • You can undo the last change with the up arrow (↑).
  • Moved images are copied to the destination folder and deleted from the source folder.
  • Deleted images are permanently removed.
  • The central panel shows the current image and you can navigate with the buttons.
  • The counter shows how many images remain.
  • Press the spacebar to open Windows Explorer at the current image.
